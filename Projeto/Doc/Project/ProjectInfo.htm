Características e operação da aplicação

Dicas

As dicas (Hints) representam uma utilidade essencial que torna o uso da aplicação muito mais prático, proporcionando ao usuário maior facilidade de uso dos recursos oferecidos pela aplicação, exibindo textos explicativos sobre as funcionalidades de cada objeto de acordo com o posicionamento do mouse sobre o objeto correspondente.

Como as teclas são tratadas internamente

ENTER

É utilizada para confirmar o conteúdo de um campo e avançar para o próximo campo. Se for o último campo, ao pressionar a tecla ENTER, o processo de gravação dos dados digitados em todos os campos será efetuado. O conteúdo de cada campo será armazenado no banco de dados da aplicação. Quando há necessidade de confirmação antes da execução de um comando, esta tecla é utilizada para confirmar.

SETAS

São utilizadas nas telas da aplicação para navegação entre os campos e na correção do conteúdo de cada campo, para posicionamento do cursor. Além de avançar para os campos seguintes com a seta para a direita ou para baixo, é também possível voltar para os campos anteriores com as setas para a esquerda e para cima.

TAB

Esta tecla é utilizada para avançar, posicionando o cursor nos próximos campos. Para voltar aos campos anteriores, pode-se pressionar a tecla TAB segurando a tecla SHIFT.

Consultas SQL

As consultas permitem uma visualização específica e análise selecionada das informações desejadas. Somente os dados procurados são exibidos, pois é feita uma triagem de informações, de acordo com os parâmetros especificados pelo próprio usuário. O usuário poderá criar suas consultas sem conhecimento da linguagem SQL, pois os recursos disponíveis já oferecem facilidade de montagem da sintaxe da consulta, facilitando o raciocínio para entender a estrutura de cada consulta. As informações são armazenadas nos campos das tabelas. Supomos que haja um campo para datas, e o usuário queira então uma consulta por data, ele terá que utilizar, nos parâmetros da consulta, o campo data desejado e os dados que forem exibidos obedecerão esses parâmetros de pesquisa.

Acessibilidade da aplicação

Esta aplicação está equipada com segurança de acesso, sendo solicitada ao usuário a entrada de uma senha para a sua utilização. O esquema de segurança utilizado na aplicação se constitui do cadastramento de Grupos de usuários com permissões de acesso a visualização, inclusão, alteração e exclusão de registros. Os usuários cadastrados sob um determinado grupo recebem as permissões designadas para aquele grupo.
